# Wedding Site

## Overview
Alison and I's wedding [website](https://ferrocoughlin.com).

## Dependencies

* Jekyll
* Bundler
* Ruby

## Running the site locally

```shell
jekyll serve --incremental
```

## Building the site

```shell
jekyll build
```
